Admin Controls

This section will go cover the Admin controls functions.

Update Profile

Upon clicking on the Update Profile button, the following interface will be displayed.


The main purpose of Update Profile is to change the password for the current logged in user.

Simply enter a new password, re-enter it for confirmation, and the password will be updated.

Add Form

Add Form allows more forms to be added.

Upon clicking on the Add Form button, the following interface will be displayed.

  • Button Placement

    • Optional, a dropdown list will be provided.

  • Input

    • Provide information for the new form, includes Name, Image source (if applicable), and list of attached buttons

    Even more information could be added by clicking on the button, located on the bottom right to the input box.

    Once finish, click Submit.

Update Form

Update Form allow modifications to be made to a form.

Upon clicking on the Update Form button, the following interface will be displayed.

  • Button placement will provide a dropdown list.

  • Enter a new name, and click Submit.

(Optional) Form name could be updated, it is turned on my default, simply turn it off by clicking on the switch.

Delete Form

Delete Form allow deletion of a form.

Upon clicking on the Delete Form button, the following interface will be displayed.

  • By default, "Deleting all of the Button" will be toggled on, allows deletion of ALL buttons related.

  • Button placement will provide a dropdown list.

  • Depends on chosen Button, more options will be provided.

Make Professor

Make Professor allows the current Admin to add more Admin, by adding an email address.

Enter the email address, and click Submit.

Make Admin

Make Admin allows the current Admin to add more Admin, by adding an email address.

Enter the email address, and click Submit.

Your Classes

Your Classes allows the Admin to view, add, and modifies classes that they are currently available.

To create a new class, enter the class name, and click Submit.

  • View a class

    • Click on the "View" button to view the class.

Create New Experiments

Upon clicking on the "Create New Experiments" button, the following interface will be displayed.

Create New Experiments allows the Professor role to create new Experiments for their classes.


Components:

    2. Class ID: This is the class ID that the Professor is creating the experiment for.


    3. Date Select: This is the date that the experiment will be held.


    4. Change Class: This is the button that allows the Professor to change the class that the experiment is being created for.

    By defalt, this option will be toggled on, simply click on "Change Class" to toggle it off.


    5. Experiment Details/Instructions: allows extra details/instructions to the experiments.


    6. Click Submit to save the changes, otherwise click on Back to return to Dashboard.

Examples

How do I change my password?
Refers to Update Profile.
How do I add a form?

  1. Within the Admin Controls, select Add Form.

  2. Enter the form name, for this example, we will choose "Indole 0000" as a form name, note that a preview will be displayed to the right of the input box.

  3. Enter the form description, for this example, we will name it "Staphylococcus aureus1243" as the form name, provide an image link, and modify the name by either the dropdown list or give it a name.

  4. Click Submit
How do I Update a form?
Refers to Update Form.
How do I Delete a form?

  1. Within the Admin Controls, select Delete Form.

  2. Select the form you want to delete, for this example, "Lactose(-) 0000" will be used.

  3. (Optional) Toggles the available options to be deleted along with the form.

  4. Click Submit
How do I view a class?
Refers to Your classes.
How do I create a new class?
Refers to Your classes.
How do I create a new experiment for my class?

  1. Within the Professor Dashboard, select Create New Experiment. Alternatively, in View Class, you could click on Create New Experiment.

  2. Fill out the title and the date period for the experiments.

  3. (Option) To change an Experiment, select a class from the available options.

  4. Provide details/instructions

  5. Click submit.
